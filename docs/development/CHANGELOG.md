# Changelog

Alle wichtigen Änderungen an diesem Projekt werden in dieser Datei dokumentiert.

## [1.0.0] - 2026-01-22

### Hinzugefügt
- Komplette Website mit 7 Sprachen (SK, EN, DE, SV, NO, ES, FR)
- Responsive Design für alle Geräte (Desktop, Tablet, Mobile)
- Dark/Light Mode mit automatischer Theme-Erkennung
- Hero-Sektion mit Tunnelbau-Bildern
- Über Uns Sektion mit Firmengeschichte
- Dienstleistungen Sektion mit 4 Hauptdiensten
- Märkte Sektion (Skandinavien, Balkan, Ukraine)
- Team Sektion mit Mitarbeiterfotos
- Projekte Sektion mit Portfolio
  - Förbifart Stockholm (21 km, TBM Operations)
  - Hammarby Sjöstad (Unterwasser Metro Station)
  - Oslo K2C (Metro Erweiterung)
  - Praha Metro D (10.6 km, Fahrerlose Metro)
  - Järfällatunnel (Stockholm Bypass)
  - TVR E05 Korsvägen (Göteborg)
- Kontaktformular mit EmailJS Integration
- Impressum & Datenschutz Seite
- SEO Optimierung
- Performance Optimierung
  - Code Splitting
  - Lazy Loading
  - Optimierte Bilder (WebP Format)
  - Gzip Kompression
  - Browser Caching

### Technischer Stack
- React 18.3.1
- Vite 6.0.11
- Tailwind CSS 3.4.17
- Framer Motion 12.0.3
- Lucide React 0.469.0
- react-i18next 15.2.2

### Tunnelbau-Methoden
- Drill & Blast Methode für die meisten Projekte
- NRTM (Neue Österreichische Tunnelbaumethode) für Praha Metro D

### Bilder
- Praha Metro D: 7 Projektbilder hinzugefügt
- Förbifart Stockholm: 21 Projektbilder
- Hammarby Sjöstad: 9 Projektbilder
- Oslo K2C: 13 Projektbilder
- Järfällatunnel: 7 Projektbilder
- Team: Schwarz-Weiß Filter auf Fotos

### UI/UX Verbesserungen
- Optimierte Navbar für kleinere Bildschirme
- Nur Flaggen-Emojis in Sprachauswahl (ohne Text)
- Kompakter Theme-Toggle (nur Icon)
- Verbesserte Emoji-Unterstützung für Edge Browser
- Zentrierte Aufzählungspunkte
- Automatisches Copyright-Jahr

### Footer
- Compliance Badges entfernt
- Katasterdienst entfernt
- Saubere Struktur

### SEO & Content
- Tunneling Service Beschreibung aktualisiert
- Erfahrungstexte angepasst
- Capabilities Liste aktualisiert (8 neue Punkte)
- TBM und NRTM Akronyme ausgeschrieben
